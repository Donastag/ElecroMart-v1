# Elecro.Mart Enterprise

Online shopping platform built with Next.js, TypeScript, and Tailwind CSS.

## Quick Start

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) to view the website.

## Features

- Product catalog with images
- Shopping cart functionality
- Responsive design
- AI-powered product recommendations

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Google Gemini AI (for image generation)
